# Team Member Images

Place your team member images in this folder with the following naming convention:

- `dr-leslie-taylor.jpg` - Dr. Leslie Taylor (Pediatrician)
- `dr-zachary-brown.jpg` - Dr. Zachary Brown (Cardiologist)
- `dr-sarah-johnson.jpg` - Dr. Sarah Johnson (Neurologist)
- `dr-michael-chen.jpg` - Dr. Michael Chen (Dermatologist)

## Image Requirements:
- Recommended size: 400x400 pixels (square)
- Format: JPG, PNG, or WebP
- Aspect ratio: 1:1 (square) for best results

## Supported formats:
- `.jpg` / `.jpeg`
- `.png`
- `.webp`

If an image is not found, a fallback with the doctor's initials will be displayed.
